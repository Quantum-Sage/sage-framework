## 🛠️ How It Works: The Hahn Echo

The core mechanism is **Dynamic Decoupling** — specifically the Hahn Echo technique.

![Hahn Echo](visualizations/04_hahn_echo_dynamic_decoupling.png)

A qubit is like a runner on a track. Without help, different qubits run at different speeds due to noise and fall out of sync. The Hahn Echo is a whistle blown halfway through the race — every runner turns around, and they all arrive at the same time.

Raw signal (red dashed): fades to near-zero at 30,000 km.
Hahn Echo protected (green solid): 100% fidelity at 30,000 km.

```python
for i in range(1, steps):
    decay = np.exp(-(t[i] - t[i-1]) / 8000)
    refocus_gain = 1.0015  # The X-pulse "turn-around"
    dd_signal[i] = dd_signal[i-1] * decay * refocus_gain
    if dd_signal[i] > 1.0: dd_signal[i] = 1.0
```

IBM's 2026 systems support `dynamical_decoupling.enable = True` — the hardware already fights for you.
