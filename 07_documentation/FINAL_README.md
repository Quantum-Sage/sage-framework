## 🛠️ How It Works: The Hahn Echo Technique

The breakthrough is **Dynamic Decoupling** — specifically the Hahn Echo.

Think of a qubit like a runner on a track. Without help, different qubits run at slightly different speeds due to noise, and quickly get out of sync. The Hahn Echo is like a **whistle blown halfway through the race telling every runner to turn around**. Because fast runners have to run further back, they arrive at the finish at the same time as slow runners. This "refocuses" the signal.

![Hahn Echo Turn-Around Trick](visualizations/Quantinuum_s_Helios.png)
*Red dashed line: raw signal fading to zero across 30,000 km. Green solid line: Hahn Echo protected identity maintaining 100% fidelity across the same distance.*

```python
# The Hahn Echo in code
dd_signal = np.ones(steps)
for i in range(1, steps):
    decay = np.exp(-(t[i] - t[i-1]) / 8000)
    refocus_gain = 1.0015  # The "turn-around" X-pulse
    dd_signal[i] = dd_signal[i-1] * decay * refocus_gain
    if dd_signal[i] > 1.0: dd_signal[i] = 1.0
```

**Why it works:**
1. **Phase Refocusing**: The X-pulse cancels static noise
2. **Environmental Shielding**: Rapid state flipping makes qubits "invisible" to interference  
3. **Real Hardware**: IBM's 2026 systems support `dynamical_decoupling.enable = True`
