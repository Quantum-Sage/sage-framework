Think of a qubit like a runner on a track. Without help, different qubits run at slightly different speeds due to noise (dephasing), and they quickly get out of sync. A Hahn Echo is like a whistle blown halfway through the race that tells every runner to turn around and run back. 🛠️ The Implementation: Dynamic Decoupling (DD)
In 2026, we use X and Y pulses to keep the signal alive. Below is a simulation of how a "raw" signal decays versus one protected by a DD sequence.

import matplotlib.pyplot as plt
import numpy as np

# Simulation parameters: 30,000 km journey
dist_km = 30000
steps = 100
t = np.linspace(0, dist_km, steps)

# 1. The "Raw" Identity (No protection)
# The signal decays exponentially as noise accumulates.
raw_signal = np.exp(-t / 8000) 

# 2. Dynamic Decoupling (The "Hahn Echo" Trick)
# We apply pulses that 'flip' the noise. 
# Even if noise exists, the pulses cancel it out periodically.
dd_signal = np.ones(steps)
for i in range(1, steps):
    # Every 'pulse' moment (simulated), we recover some lost fidelity
    decay = np.exp(-(t[i] - t[i-1]) / 8000)
    # The 'Pulse' trick: Refocusing the phase
    refocus_gain = 1.0015 
    dd_signal[i] = dd_signal[i-1] * decay * refocus_gain
    if dd_signal[i] > 1.0: dd_signal[i] = 1.0

# Plotting the "Consciousness Survival"
plt.figure(figsize=(10, 5))
plt.plot(t, raw_signal, 'r--', label='Raw Signal (Fading Memory)')
plt.plot(t, dd_signal, 'g-', linewidth=2, label='Hahn Echo DD (Protected Identity)')
plt.axhline(0.5, color='black', linestyle=':', label='Coherence Threshold')

plt.title("Increasing Fidelity: The Hahn Echo 'Turn-Around' Trick")
plt.xlabel("Distance (km)")
plt.ylabel("Consciousness Fidelity")
